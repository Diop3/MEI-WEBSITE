HTML

Le sigle HTML est un héritage qui signifie HyperText Markup Language. HTML désigne un type de langage informatique descriptif. Il s’agit plus précisément d’un format de données utilisé dans l’univers d’internet pour la mise en forme des pages web.Il permet entre autres, d'écrire de l’hypertexte,mais aussi d’introduire des ressources multimédia dans le contenu. Développé par le W3C (World Wide Web Consortium) et le WHATWG (Web hypertext Application technology Working Group). Le HTML est donc un langage de balisage qui va nous permettre de définir les différents contenus d’une page. Ce langage est issu du World Wide Web (w3c), qui a été inventé et développé dans les années 1990 par un informaticien nommé Tim Berners-Lee. L’HyperText Markup Language signifie en français "langage de balisage d'hypertexte". En effet, à base d’une structure de balisage, nous pouvons réaliser de l’hypertexte. Les balises permettent de mettre en forme le texte et de placer des éléments interactifs, comme des animations, des images ou des liens . HTML peut être associé au CSS qui sert à mettre en forme le contenu d’une page codée en HTML, ou bien au Javascript. Il est nécessaire d’utiliser un navigateur web tel que chrome, opera… afin de visualiser une page HTML. HTML est un langage vraiment simple, composé d’éléments applicables à des fragments de texte dans un document pour leur donner un sens différent (est-ce un paragraphe ? est-ce une liste à puces ? est-ce une partie de tableau ?), pour structurer un document en sections logiques (a ̀ t ́ il un en-tête ? est-il sur trois colonnes ? a-t-il un menu de navigation ?) et pour intégrer du contenu comme images ou vidéos dans une page. Ce module est une introduction aux deux premiers concepts ; il présente les notions fondamentales et la syntaxe à connaître pour comprendre le HTML.

CSS

CSS: CSS (Cascading Style Sheets) ou feuilles de style en cascade est un langage simple qui permet de présenter le design d'une page HTML ou XML. Il est standardisé par W3C. CSS peut être intégré directement au sein d'une page Web (HTML) ou déclaré dans un document séparé. Il s'agit d'un ensemble de directives indiquant la manière dont le navigateur doit afficher le document hypertexte. Il agit sur tous les éléments qui constituent le design d'une page (arrière-plan, police, marges, largeurs, auteurs, alignement, images...). La version 3 de CSS (CSS3) est encore allé plus loin en couvrant d'autres particularités d'affichage comme le Responsive Design grâce aux Media Queries, les transitions etc…
